# elementUI

> vue + vue-cli + elementUI + vue-charts

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```
# 分支
> 1.x为 elementUI 1.x版本
> 2.x为 elementUI 2.x版本

> 代码并无差别主要样式有轻微变化
